FMDisplay widget for EdgeTX
===========================

Copy the main.lua file to the radios SD card to /WIDGETS/FMDisplay/

This widget reads the current model Flight Mode and displays it on the screen in larger print with different background colors that you can edit from the Widget setup dialog.
